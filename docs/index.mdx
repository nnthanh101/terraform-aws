# terraform-aws

Registry-publishable Terraform modules with ADLC governance, 3-tier testing, and FOCUS 1.2+ FinOps compliance.

## Modules

| Module | Region | Status | Description |
|--------|--------|--------|-------------|
| [identity-center](./modules/identity-center) | us-east-1 | Sprint 1-2 | AWS IAM Identity Center (SSO) with YAML config API |
| [ecs-platform](./modules/ecs-platform) | ap-southeast-2 | Sprint 2 | ECS Fargate platform with service mesh |
| [fullstack-web](./modules/fullstack-web) | ap-southeast-2 | Sprint 2+ | Composite web application stack |

```bash
task validate       # terraform fmt + validate
```
