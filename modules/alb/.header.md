# AWS ALB Terraform Module

> Derived from [terraform-aws-modules/terraform-aws-alb](https://github.com/terraform-aws-modules/terraform-aws-alb). See [NOTICE](NOTICE.txt).

## Features

- Application and Network Load Balancer creation
- Listener management with flexible rule configuration
- Target group creation with health check customization
- Access and connection logging to S3
- Cross-zone load balancing with AZ affinity
- HTTP desync mitigation modes (monitor/defensive/strictest)
- Security group management (create or attach existing)
- WAF integration via web ACL association
- Route53 DNS record creation

## Basic Usage

```hcl
module "alb" {
  source = "github.com/nnthanh101/terraform-aws//modules/alb?ref=v2.0.0"

  name               = "my-alb"
  load_balancer_type = "application"
  vpc_id             = "vpc-1234567890"
  subnets            = ["subnet-a", "subnet-b", "subnet-c"]

  security_group_ingress_rules = {
    all_http = {
      from_port   = 80
      to_port     = 80
      ip_protocol = "tcp"
      cidr_ipv4   = "0.0.0.0/0"
    }
  }

  listeners = {
    http = {
      port     = 80
      protocol = "HTTP"
      forward = {
        target_group_key = "app"
      }
    }
  }

  target_groups = {
    app = {
      protocol = "HTTP"
      port     = 80
      target_type = "ip"
    }
  }

  tags = {
    Environment = "production"
  }
}
```
