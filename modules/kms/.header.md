# AWS KMS Terraform Module

> Derived from [terraform-aws-modules/terraform-aws-kms](https://github.com/terraform-aws-modules/terraform-aws-kms). See [NOTICE](NOTICE.txt).

## Features

- Symmetric and asymmetric KMS key creation (RSA, ECC, HMAC)
- Automatic key rotation enablement
- Custom key policy management
- Key alias creation and management
- Grant-based access control
- Multi-region replica key support
- CloudHSM custom key store integration
- External key material import support
- Configurable deletion window (7–30 days)
- Tagging for cost allocation and governance

## Basic Usage

```hcl
module "kms" {
  source = "github.com/nnthanh101/terraform-aws//modules/kms?ref=v2.0.0"

  description         = "EFS encryption key"
  enable_key_rotation = true

  aliases = ["efs/production"]

  key_administrators = [
    "arn:aws:iam::123456789012:role/Admin"
  ]

  key_users = [
    "arn:aws:iam::123456789012:role/EFSRole"
  ]

  tags = {
    Environment = "production"
  }
}
```
