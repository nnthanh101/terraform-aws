# AWS ACM Terraform Module

> Derived from [terraform-aws-modules/terraform-aws-acm](https://github.com/terraform-aws-modules/terraform-aws-acm). See [NOTICE](NOTICE.txt).

## Features

- ACM certificate creation with DNS or email validation
- Automatic Route53 DNS validation records
- Subject Alternative Names (SANs) with wildcard support
- Certificate transparency logging control
- Route53 records-only mode (for externally managed certificates)
- Validation timeout and wait controls
- Private CA (ACM PCA) integration

## Basic Usage

```hcl
module "acm" {
  source = "github.com/nnthanh101/terraform-aws//modules/acm?ref=v2.0.0"

  domain_name = "example.com"

  subject_alternative_names = [
    "*.example.com",
  ]

  zone_id = "Z2ES7B9AZ6SHAE"

  tags = {
    Environment = "production"
  }
}
```
