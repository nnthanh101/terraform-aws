# terraform-aws-efs

EFS + KMS CMK Terraform module for persistent storage with encryption at rest.
Provides SQLite/ChromaDB persistence for the xOps sovereign AI chatbot across ECS task restarts.

## Features

- EFS file system with encryption at rest (default enabled, `encrypted = true`)
- KMS CMK integration via composition pattern (pair with `modules/kms/` for key rotation)
- Single-AZ (`availability_zone_name` set, ~$6/mo) or Multi-AZ (`null` = default, HA)
- Security group with NFS port 2049 locked to specified CIDR or security group reference
- Access points with POSIX user/group mapping for container isolation
- IAM file system policy with deny-non-secure-transport (default enabled)
- Backup policy (default enabled, ENABLED state)
- Cross-region replication for RPO <= 5 min (`create_replication_configuration = true`)
- APRA CPS 234 compliant: encryption at rest, CMK key rotation, secure transport enforcement

## Architecture

```
ECS Task (SQLite / ChromaDB)
    |
    | NFS port 2049
    v
[Security Group]  (ingress 2049/tcp from specified CIDR or SG)
    |
[Mount Target]  (per subnet / AZ)
    |
[EFS File System]  encrypted=true
    |
    +<-- [KMS CMK]  (from modules/kms/, enable_key_rotation=true)
```

## Usage

### Minimal — Single-AZ ($6/mo, xOps MVP)

```hcl
module "efs" {
  source = "oceansoft/terraform-aws/aws//modules/efs"
  version = "~> 1.0"

  name                   = "xops-data"
  availability_zone_name = "ap-southeast-2a"   # single-AZ = ~$6/mo
  encrypted              = true

  mount_targets = {
    "ap-southeast-2a" = {
      subnet_id = "subnet-0aaa111"
    }
  }

  security_group_vpc_id = "vpc-0abc123"
  security_group_ingress_rules = {
    ecs_tasks = {
      description = "NFS from ECS task subnet"
      cidr_ipv4   = "10.0.1.0/24"
    }
  }

  tags = {
    CostCenter  = "platform"
    Environment = "dev"
    Project     = "xops"
  }
}
```

### With KMS CMK — APRA CPS 234 compliant

```hcl
# Step 1: create KMS key with rotation enabled (modules/kms/)
module "kms" {
  source  = "oceansoft/terraform-aws/aws//modules/kms"
  version = "~> 1.0"

  description             = "xOps EFS CMK"
  enable_key_rotation     = true
  deletion_window_in_days = 30

  tags = { Project = "xops", Environment = "production" }
}

# Step 2: pass ARN to EFS module (composition — not coupling)
module "efs" {
  source  = "oceansoft/terraform-aws/aws//modules/efs"
  version = "~> 1.0"

  name                   = "xops-data"
  availability_zone_name = "ap-southeast-2a"
  encrypted              = true
  kms_key_arn            = module.kms.key_arn   # CMK with rotation

  mount_targets = {
    "ap-southeast-2a" = {
      subnet_id = "subnet-0aaa111"
    }
  }

  security_group_vpc_id = "vpc-0abc123"
  security_group_ingress_rules = {
    ecs_tasks = {
      description = "NFS from ECS task subnet"
      cidr_ipv4   = "10.0.1.0/24"
    }
  }

  tags = {
    CostCenter  = "platform"
    Environment = "production"
    Project     = "xops"
  }
}
```

### Multi-AZ HA (no `availability_zone_name`)

```hcl
module "efs" {
  source  = "oceansoft/terraform-aws/aws//modules/efs"
  version = "~> 1.0"

  name      = "xops-data-ha"
  encrypted = true
  kms_key_arn = module.kms.key_arn

  # availability_zone_name omitted (null) = multi-AZ regional storage
  mount_targets = {
    "ap-southeast-2a" = { subnet_id = "subnet-0aaa111" }
    "ap-southeast-2b" = { subnet_id = "subnet-0bbb222" }
    "ap-southeast-2c" = { subnet_id = "subnet-0ccc333" }
  }

  security_group_vpc_id = "vpc-0abc123"
  security_group_ingress_rules = {
    ecs_tasks = {
      description = "NFS from ECS task subnets"
      cidr_ipv4   = "10.0.0.0/16"
    }
  }

  # Cross-region replication for RPO <= 5 min
  create_replication_configuration = true
  replication_configuration_destination = {
    region = "ap-southeast-4"
  }

  tags = {
    CostCenter  = "platform"
    Environment = "production"
    Project     = "xops"
  }
}
```

|------|---------|

## Outputs

| Name | Description |
|------|-------------|
| `arn` | Amazon Resource Name of the file system |
| `id` | File system ID (e.g., `fs-ccfc0d65`) |
| `dns_name` | DNS name for NFS mounting |
| `size_in_bytes` | Latest metered size in bytes |
| `mount_targets` | Map of mount targets and their attributes |
| `security_group_arn` | ARN of the security group |
| `security_group_id` | ID of the security group |
| `access_points` | Map of access points and their attributes |
| `kms_key_arn` | KMS CMK ARN used for encryption (pass-through from input) |
| `replication_configuration_destination_file_system_id` | File system ID of the replication replica |
