# AWS Web Terraform Module

> Derived from [oceansoft/alb/aws](https://app.terraform.io/app/oceansoft/registry/modules/private/oceansoft/alb/aws) v1.0.0 (Apache-2.0). See NOTICE.

## Features

- Application Load Balancer with TLS 1.3 enforcement and CPS 234 compliance defaults
- CloudFront CDN distribution (optional) with ALB origin
- WAFv2 Web ACL with AWS Managed Rules (optional)
- Route53 DNS records with automatic CloudFront/ALB alias resolution
- FOCUS 1.2+ tag propagation across all resources
- Security group with configurable ingress/egress rules

## Basic Usage

```hcl
module "web" {
  source  = "app.terraform.io/oceansoft/web/aws"
  version = "~> 1.0"

  vpc_id     = "vpc-xxx"
  subnet_ids = ["subnet-a", "subnet-b"]

  target_groups = {
    app = {
      protocol    = "HTTP"
      port        = 8080
      target_type = "ip"
      health_check = {
        enabled = true
        path    = "/health"
      }
    }
  }

  listeners = {
    http = {
      port     = 80
      protocol = "HTTP"
      redirect = {
        port        = "443"
        protocol    = "HTTPS"
        status_code = "HTTP_301"
      }
    }
    https = {
      port            = 443
      protocol        = "HTTPS"
      ssl_policy      = "ELBSecurityPolicy-TLS13-1-2-2021-06"
      certificate_arn = "arn:aws:acm:ap-southeast-2:xxx:certificate/xxx"
      forward = {
        target_group_key = "app"
      }
    }
  }

  tags = {
    CostCenter  = "platform"
    Environment = "production"
    Project     = "web"
  }
}
```
