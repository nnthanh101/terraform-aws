# AWS VPC Terraform Module

> Derived from [terraform-aws-modules/terraform-aws-vpc](https://github.com/terraform-aws-modules/terraform-aws-vpc). See [NOTICE](NOTICE.txt).

## Features

- VPC creation with customizable CIDR block and secondary CIDR support
- IPAM pool integration for dynamic IP allocation
- Public, private, database, elasticache, and redshift subnet tiers
- NAT Gateway deployment (single, one-per-AZ, or shared)
- VPN Gateway and Customer Gateway support
- VPC Flow Logs with CloudWatch or S3 destination
- DHCP options set configuration
- Network ACL management per subnet tier
- Internet Gateway and Egress-Only Internet Gateway
- DNS hostname and support configuration
- IPv6 support with custom network border groups

## Basic Usage

```hcl
module "vpc" {
  source = "github.com/nnthanh101/terraform-aws//modules/vpc?ref=v2.0.0"

  name = "my-vpc"
  cidr = "10.0.0.0/16"

  azs             = ["ap-southeast-2a", "ap-southeast-2b", "ap-southeast-2c"]
  private_subnets = ["10.0.1.0/24", "10.0.2.0/24", "10.0.3.0/24"]
  public_subnets  = ["10.0.101.0/24", "10.0.102.0/24", "10.0.103.0/24"]

  enable_nat_gateway = true
  single_nat_gateway = true

  enable_dns_hostnames = true
  enable_dns_support   = true

  tags = {
    Environment = "production"
  }
}
```
